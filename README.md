# Emotion Recognition System

## Project Description
This project is a Real-Time Emotion Recognition System using Deep Learning.

## Technologies Used
- Python
- TensorFlow
- OpenCV
- Streamlit
- NumPy

## Dataset
FER-2013 Dataset

## Features
- Real-time Face Detection
- Emotion Detection
- Live Webcam
- CNN Model
- Streamlit Dashboard

## Project Structure
- train.py
- predict.py
- app.py
- model/
- dataset/

## How to Run

1. Install requirements
pip install -r requirements.txt

2. Train Model
python train.py

3. Run Emotion Detection
python predict.py

4. Run Dashboard
streamlit run app.py
